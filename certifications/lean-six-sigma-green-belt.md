## What This Covers

### Core Methodology: DMAIC

- **Define:** Project charter, stakeholder analysis, problem statement
- **Measure:** Process mapping, data collection, baseline metrics
- **Analyze:** Root cause analysis (5 Whys, Fishbone diagrams, Pareto charts)
- **Improve:** Solution design, pilot testing, implementation planning
- **Control:** SOP development, performance metrics, sustainability measures

### Key Skills

- Process mapping and waste identification (8 forms of waste)
- Statistical analysis and data-driven decision making
- Control charts and performance metrics
- Root cause analysis techniques
- Change management and stakeholder engagement
